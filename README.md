# DOCUMENTATION

## DEPENDENCIES
`@reduxjs/toolkit`\
`react-redux`\
`axios`\
`json-server`\
`classnames`

## DEV DEPENDENCIES
`@types/axios`\
`@types/classnames`

## MAIN SCRIPTS
### `npm run client`
Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.\
The page will reload if you make edits.

### `npm run server`
Runs `json-server` on port `3001`.
